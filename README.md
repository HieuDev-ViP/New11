# HieuDev VIP Premium v7.0

Trang cá nhân tối ưu, sang trọng và gọn nhẹ cho GitHub Pages.

## Điểm nổi bật
- Nền ảnh full screen với overlay sang hơn
- Sao băng và đốm sao nhỏ chạy ngoài nền, tối ưu cho điện thoại
- Avatar nhỏ hơn, bố cục cân đối hơn
- Music player chuyển bài qua lại giữa 2 bài
- Gallery 4 ô giữ nguyên bố cục cũ
- Donate cân xứng, gọn và rõ
- Neon nhẹ, premium nhưng không rối
